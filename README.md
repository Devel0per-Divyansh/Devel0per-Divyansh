<h3>🧮 Relative Grade Calculator</h3>

<p>
<i>Late-night mini practice turned into a real academic tool.</i>
</p>

<p>
A <b>percentile-based grading system</b> that replaces rigid cutoffs with fair distribution logic.
</p>

<h4>What it does</h4>
<ul>
  <li>Upload Excel / CSV marksheets</li>
  <li>Assigns S grade to toppers (top percentile)</li>
  <li>Auto-splits remaining students into A–E</li>
  <li>Below threshold → U</li>
  <li>Visual grade distribution (SVG charts)</li>
  <li>Excel export + Dark mode</li>
</ul>

<h4>Tech Stack</h4>
<p>
HTML • CSS • Vanilla JavaScript • SheetJS • SVG<br>
<i>No backend. No frameworks.</i>
</p>

<p>
🔗 <a href="https://relative-grade.vercel.app/" target="_blank">Live Demo</a>
</p>
